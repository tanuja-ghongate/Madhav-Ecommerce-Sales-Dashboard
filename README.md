From Data to Decision: Ecommerce Analytics Dashboard is a comprehensive Power BI project built to analyze and visualize ecommerce sales data. The dashboard transforms raw transactional data into meaningful insights, helping businesses understand performance across multiple dimensions.
This project tracks key performance indicators (KPIs) such as Total Sales, Total Profit, Quantity Sold, and Average Order Value (AOV). Beyond these metrics, the dashboard provides deep insights into:

- State-wise Performance ‚Äì Identify regions driving the highest sales and profit.
- Category & Sub-category Contributions ‚Äì Evaluate product performance and profitability across categories.
- Monthly Profit Trends ‚Äì Monitor growth, seasonality, and fluctuations over time.
- Customer-wise Sales ‚Äì Discover top contributing customers and analyze buying behavior.
- Payment Mode Distribution ‚Äì Understand customer preferences in transaction methods.

The dashboard is designed with interactive visuals and slicers, allowing users to filter data by time periods, categories, or customer segments. This flexibility makes it easy to drill down into specific details or view a broader overview, depending on business needs.
